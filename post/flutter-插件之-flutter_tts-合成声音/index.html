Flutter 插件之 Flutter_tts - 合成声音

June 20, 2022

flutter_tts 是一个将文字转换为合成声音的插件，支持多语言及播放控制等

插件地址：https://pub.flutter-io.cn/packages/flutter_tts

API （version 3.3.3）介绍：

FlutterTts()                                                        # 构造

Future<dynamic> isLanguageAvailable(String language) async          # 是否支持某种语言
Future<dynamic> get getLanguages async                              # 支持的语言列表
Future<dynamic> setLanguage(String language) async                  # 设置合成语言

Future<dynamic> setSpeechRate(double rate) async                    # 速率
Future<dynamic> setVolume(double volume) async                      # 音量
Future<dynamic> setPitch(double pitch) async                        # 音调

Future<dynamic> awaitSpeakCompletion(bool awaitCompletion) async    # 等待声音播放完成
Future<dynamic> speak(String text) async                            # 播放声音
Future<dynamic> pause() async                                       # 暂停（iOS|Web）
Future<dynamic> stop() async                                        # 停止
