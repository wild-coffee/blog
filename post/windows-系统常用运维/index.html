Windows 系统常用运维

June 16, 2022

Windows查看某进程占用端口情况，或查看某端口被哪个进程占用

netstat -ano|findstr ":8080"        # 通过端口号，找到占用进程 ID，假设端口号为 8080
tasklist -V|findstr "10296"         # 找到进程 PID 后，找到对应的 exe 文件，假设 PID 为 10296
taskkill /PID 10296 -T -F           # 杀死该进程，-F 强制，-S 同时杀死子进程
