我的 Visual Studio 2013 开发环境配置

June 16, 2022

Tab 设置

Tools -> Options -> Text Editor -> C# -> Tabs

  • 选择：Smart
  • 选择：Keep tabs
  • Tab size 输入：4
  • Indent size 输入：4

黑色主题

Tools -> Options -> Environment -> General -> Color theme

行号显示

Tools -> Options -> Text Editor -> All Language：Line numbers

智能空格设置

Tools -> Options -> Text Editor -> C# -> Formatting -> Spacing

关闭：Insert space after keywords in control flow statements（if 后面的空格）

安装常用的插件

  • Productivity Power Tools 2013（安装）
  • NuGet Package Manager for Visual Studio 2013（升级）

环境和文本编辑器配置字体（等宽）

工具 -> 选项 -> 环境 -> 字体和颜色

  • 环境：Lucida Console
  • 文本编辑器：Consolas

安装英语语言包

在 微软订阅下载中心：https://my.visualstudio.com/Downloads/Featured?mkt=zh-cn 搜索 Visual Studio 2013 Language Pack 下载语言包

安装时，先关闭所有已打开的 Visual studio 软件

然后在控制台执行下载的语言包 exe 命令：{exe file path}\*.exe /Uninstall，选择 Repair

最后在 Visual studio 中切换语言

Tools -> Options -> Environment -> International Settings -> Language：English
