MacOS 使用 Repo 下载 Android 源码

December 22, 2022

使用 brew 安装 repo

brew install repo
repo --version

下载 Android 源码（以 Android 2.2 froyo 为例）

mkdir android_froyo
cd android_froyo
repo init -u git://mirrors.ustc.edu.cn/aosp/platform/manifest -b froyo
repo sync

参考资料

Android 源代码标记和细分版本: https://source.android.com/source/build-numbers.html#source-code-tags-and-builds
